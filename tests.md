# Тестирование Random Task Generator

**Автор:** Беспалова Виктория Адисовна

## Позитивные тесты
1. Запуск программы → окно открывается
2. Генерация задачи → появляется случайная задача
3. Добавление задачи → задача добавляется
4. Фильтр по типу → отображаются нужные задачи
5. Сохранение → данные записываются в JSON

## Негативные тесты
1. Пустая строка → ошибка
2. Только пробелы → ошибка
3. Нет задач в категории → сообщение

## Граничные тесты
1. Минимальная длина текста → работает
2. Длинный текст → сохраняется
3. Пустая история → программа работает

## Вывод
Все функции работают корректно.
