**Автор:** Беспалова Виктория Адисовна (замените на своё имя)

## Описание программы

Random Task Generator - это десктопное приложение с графическим интерфейсом для генерации случайных задач из разных категорий (учёба, спорт, работа). Программа помогает разнообразить日常ные дела и добавляет элемент случайности в планирование.

### Основные возможности

- ✅ **Случайная генерация задач** из предопределённого списка
- ✅ **15+ предустановленных задач** в трёх категориях
- ✅ **Добавление пользовательских задач** с проверкой на пустую строку
- ✅ **История сгенерированных задач** с временными метками
- ✅ **Фильтрация истории по типу** (учёба, спорт, работа)
- ✅ **Удаление отдельных задач** из истории
- ✅ **Очистка всей истории**
- ✅ **Автоматическое сохранение** истории в JSON
- ✅ **Цветовое кодирование** задач в истории
- ✅ **Валидация ввода** при добавлении новых задач

## Требования

- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, os, random, datetime)
